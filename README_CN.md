## 概述

本项目模拟经典弹簧-质量-阻尼系统的**自由振动**过程，系统动力学遵循二阶常微分方程：

m·x'' + c·x' + k·x = 0

用户可交互式调整物理参数（弹簧刚度、质量、阻尼系数、初始条件），并通过多组同步可视化模块实时观察系统响应——包括系统动画示意图、时域曲线图、相轨迹图、能量分析图和力分解图。

### 物理引擎

- 采用**四阶龙格-库塔法（RK4）** 数值积分，保证动力学计算的高精度
- 支持三种阻尼运动状态：
  - **欠阻尼**（ζ < 1）—— 衰减振荡运动
  - **临界阻尼**（ζ ≈ 1）—— 最快的非振荡回归平衡态
  - **过阻尼**（ζ > 1）—— 缓慢的指数回归平衡态
  - **无阻尼**（ζ = 0）—— 持续等幅振荡
- 实时计算衍生物理量：固有频率（ωₙ）、阻尼比（ζ）、阻尼频率（ωd）、衰减常数（σ）、周期（T）

### 可视化模块（5个同步画布）

| 画布名称                | 描述                                                     |
| ----------------------- | -------------------------------------------------------- |
| **系统示意图**    | 动态展示弹簧、阻尼器、质量块，包含位移箭头和弹簧力指示器 |
| **x(t) 时域曲线** | 位移-时间曲线，附带欠阻尼状态下的包络线                  |
| **相轨迹图**      | 速度-位移螺旋轨迹（轨迹颜色随时间变化）                  |
| **能量分析图**    | 动能（Ek）、势能（Ep）和总机械能随时间变化曲线           |
| **力分解图**      | 弹簧力（Fk）、阻尼力（Fc）和合力随时间变化曲线           |

### 交互式控制

- **参数滑块** — 弹簧刚度（k）、质量（m）、阻尼系数（c）、初始位移（x₀）、初始速度（v₀）
- **直接拖拽质量块** — 在系统示意图上拖动质量块设置初始位置（支持鼠标+触屏操作）
- **6种预设场景** — 轻微阻尼、临界阻尼、过阻尼、无阻尼、高频振荡、大惯性
- **播放控制** — 开始、暂停、重置、单步前进
- **速度倍率** — 0.25×、0.5×、1×、2×、4×
- **能量分布条** — 实时堆叠展示动能、势能和耗散能的分布状态

### 质量块自定义

- **自定义文本标签** — 输入任意文本（最多6个字符）显示在质量块上
- **5种文本样式** — 默认（Syne字体）、粗体、衬线体（Georgia）、等宽体（DM Mono）、表情符号
- **图片上传** — 上传本地图片替换文本，自动适配并裁剪为质量块形状

### 数据导出

- **一键CSV导出** — 包含所有记录的数据点：
  - 时间、位移、速度、加速度
  - 弹簧力、阻尼力、合力
  - 动能、势能、总能量
- 文件头部包含模拟参数，便于复现实验结果
- UTF-8 BOM编码前缀，兼容Excel打开格式

### 设计与用户体验

- **深色主题** — 搭配精心挑选的强调色板（绿色、琥珀色、靛蓝色、玫瑰色）
- **粘性侧边栏** — 滚动时控制栏始终可见
- **响应式布局** — 屏幕宽度小于800px时自动切换为单列布局
- **页面加载动画** — 分步入场的渐显动画
- **自定义排版** — Syne（标题） + DM Mono（正文）字体组合
- **纯CSS实现** — 无框架、无预处理器、无外部图标库依赖

## 技术细节

### 数值积分

模拟采用**经典四阶龙格-库塔法**，内部固定步长为0.5毫秒。动画帧率与物理计算步长解耦——每个 `requestAnimationFrame`回调会计算所需的RK4步数以覆盖实际流逝的时间（上限33毫秒，防止标签页切换时出现计算雪崩）。

```
状态向量: [x, v]
导数计算:  dx/dt = v
          dv/dt = (-k·x - c·v) / m
```

### 能量追踪

* **动能** : `Ek = ½·m·v²`
* **势能** : `Ep = ½·k·x²`
* **耗散能** : 通过数值积分累加 `Ed += c·v²·dt`（每个RK4步长计算一次）

### 画布渲染

所有五个画布均采用**设备像素比（DPR）适配缩放**——画布缓冲区尺寸设置为 `CSS宽度 × 设备像素比`，同时绘图上下文转换为CSS像素单位。确保在Retina/HiDPI显示屏上渲染清晰，无模糊线条。

`fitCanvas()`辅助函数仅在CSS尺寸实际变化时重新分配缓冲区，避免 `width: 100%`的画布元素常见的高度膨胀反馈循环问题。


## 项目结构详情

### `index.html`

语义化HTML5结构，包含两个主要区域：

* **侧边栏** (`<aside class="sidebar">`) — 粘性左侧面板，包含标题、预设、滑块、标签自定义、按钮和系统参数显示
* **主内容区** (`<main class="main">`) — 右侧内容区，包含系统画布、数据条、四个图表画布和能量条

### `style.css`

* 使用CSS自定义属性（`:root`）管理全站色彩系统
* CSS Grid实现双列页面布局、数据条和信息条
* 无 `!important`声明
* 800px媒体查询实现响应式折叠
* `@keyframes fadeUp`关键帧动画配合交错 `animation-delay`实现入场效果

### `mass_spring_damper_system.js`

代码按功能模块清晰注释，分为以下部分：

1. **标签状态与控制** — 自定义标签文本、样式和图片上传
2. **物理引擎** — RK4积分器、模拟状态机
3. **CSV导出** — Blob创建和下载触发
4. **绘图工具** — `fitCanvas()`、`roundRect()`、`drawGrid()`
5. **系统示意图渲染** — 弹簧线圈、阻尼器活塞、质量块、力的绘制
6. **图表渲染** — x(t)曲线、相轨迹、能量、力分解（四个独立函数）
7. **HUD更新** — 实时数值显示和能量条更新
8. **控制绑定** — 滑块、按钮、预设场景的事件绑定
9. **拖拽交互** — 质量块的鼠标和触屏事件处理
